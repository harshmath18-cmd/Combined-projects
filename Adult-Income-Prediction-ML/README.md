# Adult Census Income Prediction Using Machine Learning

## Project Overview
This project predicts whether an individual's annual income exceeds $50,000 using the Adult Census Income Dataset.

## Dataset
- Adult Census Income Dataset
- Source: Kaggle / UCI Repository

## Tasks Performed
1. Dataset Understanding
2. Data Cleaning
3. Feature Engineering
4. Model Building
5. Performance Evaluation

## Algorithms Used
- Logistic Regression
- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-Learn
