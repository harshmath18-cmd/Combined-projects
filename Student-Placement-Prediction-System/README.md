# 🎓 Student Placement Prediction System

A Machine Learning based Student Placement Prediction System developed using **Python, Flask, Scikit-learn, HTML, CSS, JavaScript, and Bootstrap**.

## 📌 Project Overview

This project predicts whether a student will be placed based on academic and personal details using a Machine Learning model.

## ✨ Features

- Student Placement Prediction
- Support Vector Machine (SVM) Model
- Accuracy: **86.05%**
- Confidence Score
- Attractive Responsive UI
- Flask Backend
- Render Deployment

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS
- JavaScript
- Bootstrap

## 📂 Dataset Features

- Gender
- SSC Percentage
- SSC Board
- HSC Percentage
- HSC Board
- HSC Stream
- Degree Percentage
- Degree Type
- Work Experience
- Employability Test Percentage
- MBA Percentage
- Specialisation
