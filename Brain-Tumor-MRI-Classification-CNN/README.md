# 🧠 Brain Tumor MRI Classification Using CNN

## 📌 Overview

This project classifies Brain MRI images into four categories using a custom Convolutional Neural Network (CNN).

## 📂 Dataset

Brain Tumor MRI Dataset from Kaggle

Classes:
- Glioma
- Meningioma
- No Tumor
- Pituitary

## 🚀 Features

- Custom CNN (No Transfer Learning)
- MRI Image Classification
- Data Augmentation
- Batch Normalization
- Dropout
- Confusion Matrix
- Classification Report
- ROC Curve
- Precision-Recall Curve
- Upload MRI Image for Prediction
- Confidence Score
- Prediction History
- Performance Dashboard

## 🛠 Technologies

- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Model Performance

- Training Accuracy: ~95%
- Validation Accuracy: ~92%
- Test Accuracy: ~90% (depending on training)
