# CIFAR-10 Image Classification Using CNN

## Project Overview

This project implements a Convolutional Neural Network (CNN) for image classification on the CIFAR-10 dataset.

## Dataset

The CIFAR-10 dataset contains 60,000 color images belonging to 10 categories:

* Airplane
* Automobile
* Bird
* Cat
* Deer
* Dog
* Frog
* Horse
* Ship
* Truck

## Technologies Used

* Python
* TensorFlow
* Keras
* Google Colab
* CNN (Convolutional Neural Network)

## Methodology

1. Load and preprocess the CIFAR-10 dataset.
2. Apply data augmentation.
3. Build and train a CNN model.
4. Evaluate classification accuracy.
5. Visualize results.

## Results

The CNN model achieved a classification accuracy of **85.93%** on the CIFAR-10 test dataset.

## Conclusion

The project demonstrates the effectiveness of Convolutional Neural Networks for multi-class image classification tasks.
