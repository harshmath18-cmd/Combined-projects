📖 About This Project

This project demonstrates how Machine Learning can classify Iris flowers based on four botanical measurements.

The application is developed using Python, Flask, Scikit-Learn, HTML, CSS and JavaScript.

The prediction model is trained using the famous Iris dataset with the Random Forest Classifier and provides instant predictions through a clean web interface.
